# 华容道 (Klotski) 实现计划

**目标:** 实现三国主题华容道滑块益智游戏，35 关 3 级难度，Widget-based 拖拽交互，BFS 求解器提供提示。

**架构:** Widget-based 渲染（Stack + Transform.translate + GestureDetector），参考 2048 的 AnimatedTileWidget 模式。游戏逻辑在 KlotskiBoard，关卡选择参考 Sokoban 的 ModePage 模式。BFS 求解器独立文件，预计算初始解硬编码。

**技术栈:** Flutter 3.41.5, Widget-based rendering, SharedPreferences (via ScoreService)

**设计文档:** `docs/superpowers/specs/2026-03-25-klotski-design.md`

**新增依赖:** 无

**测试模式:** 仅关键路径（Board + Solver）

**Step 3: 实现 KlotskiBoard**

核心 API：
```dart
class KlotskiBoard {
  List<Block> blocks;
  int moves = 0;
  bool get isWon;
  bool get canUndo;

  KlotskiBoard(List<BlockDef> defs);  // 从关卡定义初始化
  List<List<int>> buildOccupancyGrid();
  bool canMove(int blockId, Direction dir);
  int maxDistance(int blockId, Direction dir);
  void moveBlock(int blockId, Direction dir, [int distance = 1]);

}
```

```dart
class _BoardSnapshot {
```

**Step 3: 实现**

```dart
class KlotskiSolver {
  /// 返回从当前状态到胜利的最优解路径，null 表示无解
  static List<KlotskiMove>? solve(List<Block> blocks);

  /// 状态编码（规范化：同类型方块按位置排序）
  static String encodeState(List<Block> blocks);
}
```

BFS 流程：
- 队列存储 `(List<Block> state, List<KlotskiMove> path)`
- `Set<String>` 存已访问状态
- 对每个状态枚举所有方块 × 所有方向 × 距离 1..maxDistance → 生成子状态
- 胜利条件：曹操 block 在 (3,1)

### Task 7: Block Widget — 拖拽 + 动画  ✅

**文件:**
- 创建: `lib/klotski/klotski_block_widget.dart`

**Step 1: 实现 KlotskiBlockWidget**

核心功能：
- `StatefulWidget` with `SingleTickerProviderStateMixin`
- 1 个 `AnimationController`（吸附/弹回动画 150ms）
- `GestureDetector` 包裹：onPanStart/Update/End
- `onPanStart`：缓存四方向 maxDistance（调用 widget.maxDistance 回调）
- `onPanUpdate`：轴锁定（>8px 后锁定），偏移量夹紧到 `maxDistance * cellSize`
- `onPanEnd`：偏移 > 0.4 * cellSize → 吸附 → 调用 `widget.onMoved`；否则弹回
- `Transform.translate` 定位（baseOffset + dragOffset or animatedOffset）
- 视觉：Container + BoxDecoration（圆角、颜色按 BlockType）+ Text（汉字标签）
- 抬起效果：触碰时添加阴影 + scale 1.02

**Step 2: 验证编译**

- 点击 tile → `Navigator.pushNamed('/klotski/play', arguments: levelIndex)`

### Task 9: Game Page — 主游戏页面  ✅

**文件:**
- 创建: `lib/klotski/klotski_page.dart`

**Step 1: 实现 KlotskiPage**

参考 `sokoban_page.dart` 模式，核心区别：
- 使用 `TickerProviderStateMixin`（非 Single，需要多个 controller）
- `GameScaffold` 包裹，scoreNotifier 绑定步数
- 操作栏：撤销按钮 + 提示按钮（显示剩余次数） + 重置按钮
- 棋盘区域：`LayoutBuilder` → cellSize 计算（扣除操作栏 48px）→ `Stack`
  - `_KlotskiBoardBackground`（网格线 + 出口金色光效）
  - 所有 `KlotskiBlockWidget`（keyed by block.id）
- 提示逻辑：点击提示 → 若缓存解有效直接用，否则 BFS 求解 → 高亮方块 + 方向箭头
- 胜利检测：moveBlock 后检查 isWon → 触发胜利动画
- 胜利动画：1 个 `AnimationController`（800ms）+ `Interval` 分别驱动曹操滑出和其他方块渐隐
- 键盘支持：Focus + onKeyEvent（方向键移动选中方块，Z 撤销，R 重置，Tab 切换选中）
- 步数变色：初学/进阶始终启用，大师首次通关后启用（通过 ScoreService.hasScore 判断）
- 通关后跳转 GameOverPage（含下一关/再试一次按钮）
